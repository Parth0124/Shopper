# Shopper

## Overview

An e-commerce store for clothes with two frontends, one being for client and one being for the admin. Admin can add and view all the products whereas the client can view the product, add it to cart and even proceed to checkout. Fully responsive and attractive website.
## Live Demo

- **Client Portal**: [Shopper Client](https://shopper-swart.vercel.app/)
- **Admin Panel**: [Eat Fast Admin](https://shopper-admin-ruddy.vercel.app/)

## Features

### Client Portal

- **User Authentication**: Sign up or log in to access your account.
- **Explore Products**: Browse through the shop and products available for order.
- **Cart Management**: Add or remove items from your cart.
- **Order Placement**: Place orders.

### Admin Panel

- **Product Management**: Add new produtcs and view the list of all available products.

## Technology Stack

- **Frontend**: React, CSS, AXIOS
- **Backend**: Node.js, Express, 
- **Database**: MongoDB
- **Deployment**: Vercel (both client and admin portals), Render(for the backend)
